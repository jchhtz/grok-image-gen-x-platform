**步骤2:在X上访问Grok AI**

一旦你的账号满足所需条件,就可以继续访问Grok AI了。登录你的X账号,在屏幕左侧"主页"下的选项中点击"Grok"。或者,你可以通过点击屏幕底部搜索栏旁边的斜杠(/)图标快速访问Grok AI。这个快捷方式能立即调出Grok AI工具,让你无需浏览多个菜单就能开始互动。为了更方便,Grok在iOS和Android设备上也有独立应用。你可以从App Store或Google Play商店下载。

**步骤3:写有效的提示词**

选择Grok后,在下一个屏幕上点击"Grok 3 beta",然后选择"Try grok.com"。写提示词时,尽可能具体地描述主题、场景、风格和你想要图片传达的情感基调。例如,与其含糊地描述,不如这样写:"一个超现实主义的赛博朋克城市景观,夜晚,霓虹灯,下雨的氛围,电影角度。"这样清楚地概括了你想要的视觉元素和情绪。记住,所有生成的图片都将是JPEG格式,分辨率为1024 x 768像素。

**步骤4:生成并保存你的图片**

输入提示词后,点击"Enter"(向上箭头图标)来生成你的图片。注意只会生成两个选项,然后点击你喜欢的那个。在图片底部,你会看到"保存"。点击它将图片保存到你的电脑。你可能无法在某些工作中使用你的图片,因为所有生成的图片都有Grok水印。
